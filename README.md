# TaskManager - Laborator 1

## Scopul proiectului
Aplicația va permite utilizatorilor să gestioneze sarcinile zilnice cu ușurință.

## Funcționalități planificate
- Adăugare sarcini
- Ștergere sarcini
- Notificări

## Descriere generală
Aplicația va permite utilizatorilor să creeze și să gestioneze o listă de activități zilnice, 
pentru a-și organiza mai eficient timpul și prioritățile. Fiecare sarcină va putea fi adăugată, 
editată sau ștearsă, iar utilizatorii vor primi notificări pentru sarcinile importante.

## Tehnologii propuse
- Frontend: HTML, CSS, JavaScript (React)
- Backend: Node.js / Express
- Bază de date: MongoDB
- Control versiune: Git & GitHub
